## Project Structure

```
src/
  app/
  entities/                     # 순수 표현 컴포넌트(도메인 엔터티)
  features/                     # 데이터 fetch 등
  widgets/                      # 섹션 단위로 조합(entities + features를 조합)
  shared/
    config/                     # 정적 데이터(config) 또는 상수
    libs/                       # 유틸(예: createQueryKeys)
    ui/                         # 공통 컴포넌트
    styles/                     # scss Foundations
```
